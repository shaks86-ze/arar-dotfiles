# 🟥 UNIDAD ARAR-512 | Signalis HUD
Configuración técnica para Arch Linux + Hyprland basada en el visor de las unidades ARAR xdd.

## Requisitos
- **OS:** Arch Linux (Uso Arch Linux, no Windows)
- **CPU:** Optimizado para Intel i3-6100
- **Shell:** Zsh + Starship
- **WN:** Hyprland

## Instalación
```bash
git clone --depth=1 https://github.com/shaks86-ze/arar-dotfiles.git ~/arar-dotfiles
cd arar-dotfiles

./install.sh
