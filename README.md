# 🟥 UNIDAD ARAR-512 | Signalis HUD
Configuración técnica para Arch Linux basada en el visor de las unidades ARAR xdd.

## Requisitos
- **OS:** Arch Linux (Uso Arch Linux, no Windows)
- **CPU:** Optimizado para Intel i3-6100
- **Shell:** Zsh + Starship

## Instalación
```bash
git clone [https://github.com/tu-usuario/arar-dotfiles.git](https://github.com/tu-usuario/arar-dotfiles.git)
cd arar-dotfiles
chmod +x install.sh
./install.sh
